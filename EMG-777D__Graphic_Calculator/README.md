# EMG-777D graphical calculator

Wikipedia (hu): https://hu.wikipedia.org/wiki/EMG-777

## CPU card
12x 82S191 (2k x 8, bipolar PROM), total: 24 kByte on CPU  
![CPU card](CPU/CPU.jpg | width = 100)

## ROM card
16x 2732A (4k x 8 EPROM), total: 64 kByte on SYSROM  
8 ICs on left side  
8 ICs on right side (on second address bus)  
![SYSROM card](SYSROM/SYSROM.jpg | width = 100)
